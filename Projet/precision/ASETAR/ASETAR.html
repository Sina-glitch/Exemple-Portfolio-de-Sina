Description du projet:

ASETAR est une organisation lycéenne souhaitant obtenir un site.
Sur ce site, elle veut pouvoir :

- Afficher les derniers évènements du lycée
- Enregistrer et modifier des profils
- Mettre des produits à vendre à l'effigie du lycée et des croissants
- Mettre les informations générales du lycée

Trois types d'utilisateurs sont amenés à utiliser cette application :

- Une partie administrateur, pour les membres du personnel
- Une partie utilisateur, pour les visiteurs du site
- Une partie élève, pour les élèves du lycée
