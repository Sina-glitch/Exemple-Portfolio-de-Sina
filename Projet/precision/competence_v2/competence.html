Description du projet:

La société Egnom souhaiterait mettre en place une solution logicielle pour gérer et développer les compétences
de ses collaborateurs.

La solution logicielle doit permettre de mettre en relief le portefeuille de compétence de l'entreprise avec les
compétences réellement acquises par ses collaborateurs.

Pour que les collaborateurs valident des compétences, les administrateurs RH se chargeront de créer des
évaluations avec les différents critères qui les composent.
Ces critères seront définis par :
- Un intitulé
- Un nombre de points
- Aucune, une ou plusieurs compétences associées

Puis, une fois créé, c'est également les administrateurs RH qui se chargeront de remplir les informations sur
les évaluations pour chaque collaborateur.
Les informations suivantes pourront être reportées :

Pour les différents critères :
- Le nombre de points obtenus
- Une remarque

Pour l'évaluation :
- Une remarque générale

De leur côté, les collaborateurs pourront consulter dans le détail, les résultats obtenus sur ces évaluations.
Dans un tableau global, le degré de maîtrise de chaque compétence sera défini automatiquement en fonction de la
moyenne des points obtenus dans chaque critère :
- 0% à 33% -> Non acquise
- 33% à 66% -> Partiellement acquise
- 66% à 100% -> Acquise
