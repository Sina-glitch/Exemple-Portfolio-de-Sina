Description du projet:

La société Egnom utilise une solution open source nommée Dolibarr.
Cette solution open source permet de gérer des données via l'interface graphique ou une API.
La société souhaite donc développer une solution spécifique pour rendre plus performant le processus de gestion de remboursement.
Cette solution sera développée sous la forme d'une application.
